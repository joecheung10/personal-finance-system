# personal-finance-system
Type: Currency, Account, Transaction, Budget, Category
Use Case:
1. Add Currency
2. Add Account (Wallet, Bank, E-Wallet)
3. Add Category
3. Record Transaction (Income, Expenditure, Transfer)
4. Record Investment
5. View The Calendar Page which can see the transaction and investment record
6. View The Account Balance Page
7. View The Analyze Page which consist pie chart of income, expenditure group by category, MoM line chart of the total balance, income, expenditure
